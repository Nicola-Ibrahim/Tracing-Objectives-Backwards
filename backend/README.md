# 🚀 Backend: Inverse Mapping Engine

This is the FastAPI-powered backend for **Tracing Objectives Backwards**. It handles dataset generation, surrogate model training (GPBI, MDN, etc.), and generative diagnostics.

## 🚦 Quick Start (Local Development)

### Prerequisites
- [uv](https://github.com/astral-sh/uv) (Fast Python dependency manager)

### Installation
```bash
uv sync
```

### Running the API
```bash
uv run fastapi dev src/main.py
```
